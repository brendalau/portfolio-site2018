Measure World

{ Processing / Adobe Illustrator }

An interactive learning game focused on increasing familiarity with Chinese measure words. Developed as a final project for my Programming Basics course.

In the Chinese language, when you describe a noun or something conceptually tangible you need to first define its “measure”. In English, using “a” or “the” can usually suffice. There are different measure words for all sorts of nouns to help classify the thing you are describing. With all the different nouns that exist, though, you can only imagine how many measure words also exist. Some measure words can also be applied to nouns that follow a certain description.

What I mean by this is that with the measure word 张, for example, the application can be for any long, flat object like paper or CD’s. So a clear problem that arises from this is that it can be a serious struggle to learn the associations and proper usages for these MW for experienced speakers as well as non-native speakers (who are not accustomed to hearing Chinese regularly) especially.

The main objective of this game then is to support active engagement with these measure word and noun connections through drag-and-drop sorting. Sorting boxes and noun items are randomly generated each game. The number of incorrect attempts is hidden from the user until the end in order to not discourage progress. The end screen also features the incorrectly paired noun items with their correct corresponding measures to further promote learning.
